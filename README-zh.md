## ✨ 功能特性

✅ **变量 / 函数 / 类名混淆** —— 保留 API 接口，避免破坏调用  
✅ **导入与关键字保护** —— 确保混淆后的代码可运行  
✅ **AST 语法树级别处理** —— 非简单文本替换  
✅ **文件上传与在线处理** —— 粘贴或上传即可混淆  
✅ **错误检测与提示** —— 若代码存在语法错误会直接提示  

---

## 🚀 使用方式

1. 打开网站主页（地址可向作者获取）  
2. 粘贴或上传 Python 源码  
3. 点击 **Obfuscate（混淆）**  
4. 下载混淆后的代码文件
